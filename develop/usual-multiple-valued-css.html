常用的必须多参数才可以看到效果以及可以多参数叠加的css

发表于 2014-10-14   |   分类于 develop   |     |   阅读次数

随着css3的出现，很多原本要借助js实现的效果都可以用css3来实现了。那么现在就来小看一下，有哪些css是必须要多参数（>=2个参数）才会有效果的，同时又有哪些css3属性又是可以多值叠加使用的。

首先来说一说，那些必须要多参数才可以有作用效果的css属性。

border系列，包括border-top/right/bottom/left

box-shadow

none：无阴影
第1个长度值用来设置对象的阴影水平偏移值。可以为负值
第2个长度值用来设置对象的阴影垂直偏移值。可以为负值
如果提供了第3个长度值则用来设置对象的阴影模糊值。不允许负值
如果提供了第4个长度值则用来设置对象的阴影外延值。可以为负值
color：设置对象的阴影的颜色。
inset：设置对象的阴影类型为内阴影。该值为空时，则对象的阴影类型为外阴影

text-shadow

这个跟box-shadow也有点像。

none：无阴影
第1个长度值用来设置对象的阴影水平偏移值。可以为负值
第2个长度值用来设置对象的阴影垂直偏移值。可以为负值
如果提供了第3个长度值则用来设置对象的阴影模糊值。不允许负值
color：设置对象的阴影的颜色。

必须多个参数的暂时就只想起了那么些了，剩下的就是一些可叠加使用的css3属性了。通过那些属性值的叠加也可以做出一些非常好看好玩的东西。

其实常见的就是那些背景、文字、阴影、动画了。这个的话，直接看别人的作品似乎会来得更有视觉冲击力也好模仿和练习研究。
