又可以静静的学习研究和写代码玩了

发表于 2016-06-06   |   分类于 diary   |     |   阅读次数

往往复复后又回来了，只是这次的回来更多了一份安静吧。感觉生活并不缺什么所以也就不会刻意的去想要什么，当然除了还缺一个人以外。现在会去想要找一个人，一方面是自己确实想要找个人，还有一方面是来自奶奶那一代人的因素，可能主要就是这个因素才会让我比较有紧迫感吧。

生活不可能一帆风顺，不管是事业还是情感，所以会不断调整自己坦然处之。你们的热闹是你们的，不是我的跟我无关我也不会想要去主动关注。我现在都已经把微信推送都给关闭了，直到某天或许会因为谁或者某些事情再重新开启吧。

现在已经是6月6号了，一年都快要过去一半了。上半年有了一些变动，可能下半年还更会有一些变动吧。不会一直让信任自己的人失望的，也不会让看重自己的人苦苦等待的。以前不敢的一些事情和行为，现在内心却无比的想要去跃跃欲试。自我感觉一直还算是一个比较喜欢热闹的人，但更是一个愿意且能享受安静的人。

生活是需要希望的，也需要为生活而奋斗的激情与韧性。不管你现在的生活是否一团糟亦感觉生活郁郁不得志、大龄单身狗、房奴、月光族等等，不管你愿不愿意你现在就是活在了柴米油盐酱醋茶的红尘世俗社会里。

很多时候，只想一个人静静，喜欢心理学的我很多事情其实一看就懂的，只是不想去想烦心的事，想让心空空的，我甚至更喜欢做个正常的傻瓜。不知道从什么时候开始，恋上了孤独。你一直觉得很孤单，心里藏着许多伤口，你害怕被误解所以一直掩盖着，直到你知道其他人心里其实也有伤口。于是你把伤口展露出来，希望以此寻获同类，但最终你会失望的发现，每个人的伤口都不相同，有深有浅，谁也安慰不了别人，理解不了别人，你只好沉默着把它们再藏起来。于是，你变得比从前还要孤单。

人生总要疯狂个几回，不然，以后，或许就没机会了。你可知，有些错过其实就是一辈子的擦肩而过。我以前会因为各种因素而害怕一段情感的开始，但是我现在却渴望开始“打怪升级闯关积累经验”之旅。

今天也已经粗略的从新把博客过了一遍，换了一套新皮肤。博客皮肤啥的，想想这毕竟不是自己擅长的东西，我还是回归思考专注分析比较符合自己的专长。然后好好的把欠下的系列文章给补上，不然你怎么开始下一份新旅程新阶段呢。

给自己一个新的憧憬，好好赚钱然后环游世界；给自己一个新的洗礼，好好阅读然后提升自己；给自己一个新的目标，好好学习然后产出变现。

以前一直在不断的给自己加东西，而今却不断的给自己减东西，更不想被无关无趣的人打扰。现在大部分的时间，这段时间每天起来的第一件事就是：今天我应该要完成哪些，然后赶紧出个demo和做个样品出来玩下。
