准备再次出发！

发表于 2012-06-25   |   分类于 diary   |     |   阅读次数

这个学期结束的比较早，我十九号就到家了。

今天是25号，现在也已经买好了28号的火车票了。父母已经在厦门旅游了，我自己一个人在家守家吧（因为平时父母都走不了，要做生意）！然后，久违的自己做饭炒菜等等的经历又再次出现，现在发现有些的确是不熟练。不过话说回来了，那些简单的炒炒菜什么的，应付自己一日三餐还是没问题的。

在家的这些天，也找了些同学问问，聊聊天！发现，和有些人的思想开始有距离了。不同的人，面对同一件事情，有些回答没挑战性，有些说很难。这个，就像是我们小学课本上的小马过河吧！还是回归到自己的客观现实中，实事求是。

对于梦想，对于追求，如果有条件的话，应该毫无疑问的义无反顾的坚持下去，直到结束了！也许，结果会出乎意料，但过程很重要，成长很重要！生活中，不会你想象的那么简单，但也不会是你想象的那么难。

未来的道路方向，和父母亲友的想法是有些冲突的。现在，无非就是两条路吧！一条算是我想要的路，按着自己的计划前进，收获自信，过有意义的人生；一条算是客观现实的路，毕业了马上成家、工作，然后亲友甚至都开始帮我介绍女朋友了！生活中，似乎父母也想要有孩子陪在他们身边了。有些情况，似乎真的不是三言两语就可以改变的，这跟环境有关系。

六月是一个毕业的季节。生涯中，感觉自己经历了不少“毕业季”。

有时候，我真的很想说，我们的相遇就是一种缘分。在真正的等到毕业季的时刻，那些阻碍人们生活的小琐事甚至是一些“勾心斗角”都变得如此的渺小和不堪一击。就像曾经的毕业季离别，那些压抑了太久的情感最后都会爆发出来。平常很少见的男女间的拥抱，变得再平常不过了。曾经很陌生的同学，似乎一下子就变得如此的熟悉和不舍起来。

在离别的时候，总是少不了一些问候珍重等等。然后，有些人因为惧怕离别而选择“消失”，有些人则在不断的寻找搜索着属于那些过去岁月的痕迹，那些有关的照片和视频。甚至，有些人真的会发觉很后悔，为什么曾经不多拍点照片呢！！！

生活中，好朋友，好同学还是挺多的！很多人也是重情重义的，不会跟你勾心斗角的，不会跟你斤斤计较，也没有那么多的世俗之见。同学朋友之间只有到一些特定的时刻，有些情感才会越发的强烈。有真感情的同学朋友之间，虽然可能会因为时间和距离的问题感觉情感上淡了，但真到相遇的时刻一切又都变得如此的熟悉。

有些人我会想念，有些人我很热意的为ta付出，有些人我会记住。这个时刻，也许各自都在忙碌着准备着什么吧，我也在忙碌。等到六月份交的最后一个月的校园网断网后，我不续费了。什么时刻都可以做的事情，为什么就不把它延后一些来呢！最后一学年，目标单纯又单一。然后，希望在离校前的两三个月内，我可以做个背包客，去一些地方，同时也拜访一些同学！

人的成长，要想改变些什么呢？思想观念、生活习惯、形态意识、思维逻辑、优良美德等等都是一些内在的东西、、、
